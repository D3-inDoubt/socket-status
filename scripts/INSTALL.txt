- IP-Tables to permit outgoing connections to the target IP & Port

Salve!
Dies ist ein simples Script um den Socket-Status eines Nodes zu prüfen und via einer einfachen Bild / Text Ausgabe anzuzeigen.
Zuerst muß in Deinem php Script die Funktion eingefügt werden.
Diese kann an einer beliebigen Stelle stehen, vorzugsweise im Header Bereich.

Als nächstes kommt die Abfrage für den gewünschten Node (IP, PORT).
Ergänze bei der Abfrage den Wert IP mit der IP-Adresse bzw. dem Hostnamen.
PORT ergänze mit dem Port der geprüft werden soll. Eine Angabe ob TCP oder UDP ist nicht notwendig.

Wie komme ich an die gewünschte IP / Port?
Der einfachste Weg geht über die OS Bordmittel.

Windows:
Verbinde Dich mit Deinem Spieleclient auf den gewünschten Server!
Öffne als Administrator die cmd.exe
Gib in der Kommandozeile 'netstat -b' ein.
In der Zeile oberhalb von TERA.exe steht die IP / Port des gew. Servers.

Salve!
This is a simple script to check the socket-status of an node.
First you need to add the php function into your php file

Then replace the IP and Port values with the original ip-adress / hostname and port

To check with IP / Port you need you can use the OS boardtools.
The easiest way is via netstat. Open the Gameclient, open the cmd.exe under windows as admin.
Into the shell type 'netstat -b' and search for TERA.exe
